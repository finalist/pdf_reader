= PDF Reader plugin

Gem that gives you the "power" to extract raw text from a PDF.

== Installation

Type:
  sudo gem install finalist-pdf_reader

And... done!

== Usage

Create a new object and call +raw_text+:
  p = PDFReader.new('test.pdf')
  puts p.raw_text

... Done!
