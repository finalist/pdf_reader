= PDF Reader plugin

Rails plugin that gives you the "power" to extract raw text from a PDF.

== Installation

Type:
  ./script/plugin install git://github.com/finalist/pdf_reader.git

And... done!

== Usage

Create a new object and call +raw_text+:
  p = PDFReader.new('test.pdf')
  puts p.raw_text

... Done!
